# 基础风格

MultiPixel乘方像素是一款基于新版Minecraft纹理的32×分辨率原版高清风格纹理包，如果您喜欢新版纹理，并想提升其分辨率，那么这款纹理包可能对您胃口。

# 乘方像素的高清风格

边缘线和一些杂色纹理会被细化，但对于整个颜色区域的相似色块纹理则会采取柔和的高清处理方式，而不会添加过多细节。
